# Tecnologies used
1. Next.js (SSR)
    -> Next Router
    -> Next image
2. Redux
3. Tailwind CSS
4. NextAuth Authentication
5. FakeStoreApi (For products)
6. Heroicons svg icons
7. Webhooks
8. Stripe Checkout
9. Firebase

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
